# Weather App

A beautiful weather checking website using OpenWeatherMap API.

## Setup

1. Clone this repository
2. Copy `config.template.js` to `config.js`
3. Get your free API key from [openweathermap.org](https://openweathermap.org/api)
4. Paste your API key in `config.js`
5. Open `index.html` in browser

## Files

- `index.html` - Main structure
- `style.css` - Styling with glassmorphism effects
- `script.js` - Weather logic and API calls
- `config.js` - API configuration (not in repo, create from template)
